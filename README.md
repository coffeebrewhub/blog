# BrewHub.Coffee

☕️ A blog for the BrewHub Coffee community.

## Created with

- [Elm](https://elm-lang.org)
- [Elm Pages](https://elm-pages.com/)
- [Tailwind](https://tailwindcss.com/)
- [Browser Sync](https://browsersync.io/)

## Installation

Install node modules

```sh
$ npm i
$ npm run postinstall
```

## Running

Just use

```sh
$ npm run start
```
